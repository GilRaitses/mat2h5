magat fairy

docs opens repo about page
magat fairy transforms MAGAT analyzer data from MATLAB format to HDF5 for Python workflows

convert data

Convert MAGAT experiment data from MATLAB structured format into Python-ready H5 files. Drag and drop a data folder (experiment, ESET, or collection); detection and conversion run automatically. Outputs are saved to the exports/ folder in the repository.

documentation

Documentation for magat fairy covers supported data formats, conversion options, output structure, and troubleshooting. Supported inputs include single experiments, single tracks, ESETs, and collections with multiple ESETs.

examples

Example workflows and sample data show how to convert different data types, explore output H5 structure, and apply best practices for working with converted MAGAT data in Python.

about

magat fairy transforms MAGAT analyzer data from MATLAB format to HDF5 for Python workflows. It detects data types, handles calibration, and preserves experiment metadata to bridge MATLAB and Python ecosystems.
